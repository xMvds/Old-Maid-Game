Hoe speel je?
Old Maid • 1 lobby • host is alleen voor dev
  1. Start de server op 1 laptop/pc: npm install → npm start.
  2. Iedereen opent dezelfde link naar de server (op mobiel geen localhost, maar het IP-adres van de host-pc).
  3. Vul je naam in → Join tafel.
  4. De host start het spel via /host.
  5. Ben jij aan de beurt? Je krijgt automatisch het kaart-keuze scherm en kiest 1 kaart van je buur (verborgen backs).
  6. Pakt iemand een kaart van jou? Tik 1 van je kaarten om die omhoog te zetten (bluff/tell).
Als iemand op een ‘lege tafel’ belandt, zit die waarschijnlijk op een ander adres. Zorg dat iedereen exact hetzelfde adres gebruikt.
